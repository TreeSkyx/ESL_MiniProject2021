### Project : Automatic Contactless Alcohol Dispenser

### Features
- Contactless
- Body Temperature Measuring
- Line Notification
- Easy to Move & Refillable

### Equipment
    - ESP32
    - Mini pump
    - PNP transistor
    - Infrared tracking sensor
    - Infrared temperature sensor
    - Breadboard
    - Li-ion 18650 battery
    - Two-ways slide switch
